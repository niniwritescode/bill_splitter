# Bill Splitter App

A simple web application to split bills among friends.  
Built with **Python (Flask)** for the backend and **HTML, CSS, JavaScript** for the frontend.

---

## 🚀 Features
- Enter total bill amount and number of people
- Calculate each person’s share instantly
- Clean and simple UI
- Flask backend for handling logic
- Responsive design with HTML/CSS/JS
